# Ruta de Aprendizaje en Ingeniería de Software

## Descripción
Este repositorio documenta un programa de formación autodidacta 
de 24 semanas, enfocado en adquirir las habilidades fundamentales 
del desarrollo de software moderno.

## Contenido del programa
- **Fundamentos:** Python, Programación Orientada a Objetos
- **Datos:** Estructuras de datos, Bases de datos relacionales
- **Herramientas:** Control de versiones con Git y GitHub  
- **IA aplicada:** Introducción a Machine Learning

## Objetivo
Alcanzar el nivel técnico requerido para desempeñarme 
profesionalmente como Ingeniero de Software.
